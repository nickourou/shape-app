🟢 ΟΔΗΓΙΕΣ

1. Πήγαινε στον φάκελο `client/` και βάλε εκεί τον React κώδικά σου (App.jsx, index.html, κλπ)
2. Μέσα στον `client/`, τρέξε `npm install` και μετά `npm run build` (θα φτιάξει φάκελο `dist/`)
3. Πήγαινε στον φάκελο `server/` και τρέξε `npm install`
4. Τρέξε `npm start` και η εφαρμογή θα είναι διαθέσιμη στο http://localhost:3000
5. Για deploy σε Render, ανέβασε όλο τον φάκελο με backend (Express) ως Node.js app

🚀 Με αυτό το setup λειτουργεί πλήρως και το Telegram login, χωρίς περιορισμούς CSP!
